Lung & Bronchus Cancer mortality — ML dashboard

Explore county-level modeling, training workflows, and variable importance through an interactive Streamlit app, or open the bundled static HTML prototype in your browser.

Run Streamlit Open LBC HTML SQL engine site

What’s included

  • Streamlit dashboard — EDA, splits, models, metrics, training, and importance views under LBC_Dashboard/lbc_dashboard/ (or legacy lbc_dashboard/ paths in older clones).
  • SQL Engine Benchmark — DuckDB · Polars · SQLAlchemy · optional PySpark, Dask-SQL, psycopg2 — landing at sql_engine_benchmark/ and static demo sql_engine_benchmark.html, app under SQL_Engine_Benchmark_Dashboard/.
  • LBC standalone page — lbc_dashboard.html at the repo root for quick viewing without Python.
  • This site — minimal HTML, CSS, and JS in web/ published as GitHub Pages (workflow copies static demos and SQL landing into _site).

Run the Streamlit app

From the lbc_dashboard directory:

pip install -r requirements.txt
streamlit run app.py

Then open http://localhost:8501 in your browser.
