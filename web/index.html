Lung & Bronchus Cancer mortality — ML dashboard

Explore county-level modeling, training workflows, and variable importance through an interactive Streamlit app, or open the bundled static HTML prototype in your browser.

Run Streamlit Open static HTML

What’s included

  • Streamlit dashboard — EDA, splits, models, metrics, training, and importance views under lbc_dashboard/.
  • Standalone page — lbc_dashboard.html at the repo root for quick viewing without Python.
  • This site — minimal HTML, CSS, and JS in web/ as a template you can extend or host (e.g. GitHub Pages with the web folder as the publish root).

Run the Streamlit app

From the lbc_dashboard directory:

pip install -r requirements.txt
streamlit run app.py

Then open http://localhost:8501 in your browser.
