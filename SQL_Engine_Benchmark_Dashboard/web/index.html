⚡ Benchmark — SQL query

From sidebar template · then ▶ Run Benchmark. Below it, Custom query · ▶ Run Custom Query (independent SQL). Placeholders {table} and {lookup_table} are filled automatically.

Example (“DateTime: Extract hour…”) · Yellow Taxi columns: tpep_pickup_datetime, fare_amount, … 

SELECT CAST(strftime('%H', tpep_pickup_datetime) AS INTEGER) AS pickup_hour,
       COUNT(*) AS trip_count
FROM trips
GROUP BY pickup_hour
ORDER BY pickup_hour

Templates prefer strftime-style SQL so SQLAlchemy (SQLite) matches DuckDB / Polars for common demos.
