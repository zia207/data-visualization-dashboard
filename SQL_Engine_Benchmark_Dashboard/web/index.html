Engine reference
  • DuckDB — embedded OLAP, fast local analytics.
  • Polars — Rust DataFrames + SQL context.
  • SQLAlchemy — SQLite in-memory in this dashboard.
  • PySpark — needs Java; cluster-scale.
  • Dask-SQL — parallel pandas SQL.
  • psycopg2 — PostgreSQL (needs a server).

What’s included

  • Streamlit app — SQL_Engine_Benchmark_Dashboard/app.py · leaderboard, execution log, “how to fix” hints, Query & Visualize charts, taxi / tax templates, uploads.
  • Engines at a glance — one‑line blurbs plus optional installs (PySpark needs Java, psycopg2 needs Postgres for real runs).
  • Soft theme — .streamlit/config.toml + custom CSS aligned with these pages.
  • This site — static landing next to the app; GitHub Pages path /sql_engine_benchmark/ when wired in CI.
  • Standalone chart demo — repo root sql_engine_benchmark.html for a browser‑only teaser.
