Analytics & benchmarking

SQL engines on the same query & dataset

Run preset or custom SQL against DuckDB, Polars, SQLAlchemy (SQLite), optional PySpark, Dask-SQL, and psycopg2 — with timing charts, Yellow Taxi or NYS Tax built-ins, and joins to a zone lookup CSV when you need them.

Run Streamlit Open static HTML demo

What’s included

  • Streamlit app — SQL_Engine_Benchmark_Dashboard/app.py · benchmark tab, query & viz, data preview, schema, engine guide, and “how to fix” hints when an engine fails.
  • Built-in data — synthetic Yellow Taxi-style trips (shared templates) or NYS Tax sample; upload CSV / Parquet anytime.
  • Static demo page — sql_engine_benchmark.html at the repo root for a quick in-browser preview (no Python), published next to the LBC static demo on GitHub Pages.
  • This landing page — same layout pattern as web/index.html in the repo, kept under this folder so documentation stays next to the Streamlit app.

Run the Streamlit app locally

From the SQL_Engine_Benchmark_Dashboard directory:

pip install -r requirements.txt
streamlit run app.py

Then open http://localhost:8501. Optional theme: .streamlit/config.toml.
